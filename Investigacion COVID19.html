$7.$ Este paso es necesario realizarlo, ya que el metodo de preprocesamiento nos ayuda a combinar las variables categoricas con las numericas.

Para transformar las categoricas a numericas se utiliza el metodo OneHotEncoder, y para separar las numericsa se utiliza el SantardScale, esto se realiza debido a que ciertos campos de texto no se van a correlacionar con campos numericos, y viceversa un numerico no se puede correlacionar con un texto.
